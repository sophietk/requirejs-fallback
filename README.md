requirejs-fallback
==================

Tiny app to test a requirejs dependency fallback, also in the minified app.

### Installation
```
npm install
bower install
```

### Dev version
```
grunt serve
```
Then go to [localhost:9000](http://localhost:9000)

### Minified version
```
grunt serve:dist
```
Then go to [localhost:9005](http://localhost:9005)

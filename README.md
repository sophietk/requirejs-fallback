requirejs-fallback
==================

Tiny app to test a requirejs dependency fallback, also in the minified app.

### Installation
```
npm install
bower install
```

### Dev version
```
grunt serve
```
Then go to [localhost:7000](http://localhost:7000)

### Minified version
```
grunt serve:dist
```
Then go to [localhost:7001](http://localhost:7001)
